Assault

What is Assault?

Assault is a highly customizable handcrafted battery indicator with support for a wide range of window managers, docks, and trays.

Will it work for me?

Our goal is to support as many WMs, docks, trays, and bars as possible. Is yours not listed? File a feature request or fork it and send a pull request.

  • WindowMaker DockApp
  • lxpanel
  • AwesomeWM via Awful (coming soon)

Customization

Almost everything is configurable. You can change the size of the battery, the size of the lightning bolt, the size of the text, the colors, even down to the size of the peg on the side.

Every interface has its own way of exposing configuration options, but the options it exposes are the same, excluding the generic StatusIcon interface, which determines dimensions automatically.

Battery Height
The height of the inside of the battery
Battery width
The width of the inside of the battery
Bolt Width
The width of the charging "bolt" indicator
Bolt Height
The height of the charging "bolt" indicator
Peg Height
The height of that peg on the side of the battery
Peg Width
The width of that peg on the side of the battery
Stroke Width
The thickness of the battery's outline
Font Face
The font face of the indicator text
Font Size
The font size of the indicator text
Update Interval
How often the indicator will update
Critical Level
The level at which the battery is considered "critical"
Discharging Color
The color of the indicator when discharging, in RGBA
Critical Color
The color of the indicator when below the critical level, in RGBA
Charging Color
The color of the indicator when charging or on AC power, in RGBA
Background Color
The color behind the indicator or widget, in RGBA
